# UI-Testing-Automation

## Built with

```
- Javascript
- cypress
```

## How to run :

- Clone Repository
- 
```
[https://github.com/triani16/UI-Testing-Automation.git]
```

## Install the project
```
- npm i
- npm i cypress
```
  
## Run the project
```
- npx cypress open
- npx cypress run
```
